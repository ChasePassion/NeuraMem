# AI Memory System 可视化演示

基于 Gradio 的 AI 记忆系统可视化测试界面。

## 功能特性

- **左侧面板**: 实时显示记忆库状态（情景记忆 + 语义记忆）
- **右侧面板**: 对话测试界面
- **记忆巩固**: 支持手动触发巩固流程，实时显示处理进度
- **记忆管理**: 支持刷新、重置记忆

## 安装依赖

```bash
pip install gradio>=4.0.0
```

## 运行方式

```bash
# 在项目根目录下运行
python demo/app.py
```

然后在浏览器中访问: http://localhost:7860

## 使用说明

1. **初始化系统**: 输入用户ID，点击"初始化系统"
2. **发送消息**: 在右侧对话框输入内容，系统会自动判断是否存储为记忆
3. **查看记忆**: 左侧面板实时显示所有记忆
4. **运行巩固**: 点击"运行巩固"执行语义提取操作
5. **清空记忆**: 点击"清空记忆"删除当前用户所有记忆

## 测试建议

- 个人信息: "我是北京大学计算机专业的学生"
- 明确记忆: "请记住我喜欢喝咖啡"
- 项目信息: "我正在开发一个AI记忆系统"
- 闲聊测试: "你好" (不会被记录)
